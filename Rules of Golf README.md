A walk through golf's rule-making in five snapshots. The rolling [[Rules of Golf]] file tracks the current code; git history holds all 67 versions from 1744 to 2023. The files below are pinned copies of representative codes, kept readable without git archaeology.

## Pins

- [[Rules of Golf 1744]] — Leith, Gentlemen Golfers. The 13 original Articles; the foundational document.
- [[Rules of Golf 1858]] — R&A, first code issued as governing body. The hinge from one club's rules to a wider authority.
- [[Rules of Golf 1891]] — R&A, last 19th-century revision. The closest reference for what the 1899 unified code was consolidating.
- [[Rules of Golf 1925]] — R&A, peak hickory era. Last code before steel-shaft adoption reshaped the game.
- [[Rules of Golf 1960]] — R&A / USGA, settled joint code at mid-century.

Source: <https://www.ruleshistory.com/> — Peter Lewis's archive of every printed code.
